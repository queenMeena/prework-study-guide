Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • <div>: Used for grouping and styling elements in HTML.
  • <p>: Represents a paragraph of text in HTML.
  • <a>: Creates hyperlinks to other web pages or resources.
  • <img>: Embeds images into a web page.
  • <ul>: Defines an unordered list, typically used with <li> for list items.

CSS

  • Selectors: Identify HTML elements to apply styles in CSS.
  • Properties: Define specific styles (e.g., color, font-size) for elements.
  • Values: Specify the settings for properties (e.g., red, 16px).
  • Classes: Reusable CSS styles that can be applied to multiple elements.
  • Box Model: Describes an element's layout (margin, border, padding, content).

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • Repository: A storage space for your Git project and its history.
  • Commit: A snapshot of your project's current state, recorded in Git's history.
  • Branch: A separate line of development for new features or bug fixes.
  • Pull Request: A request to merge changes from one branch into another.
  • Clone: Creating a local copy of a remote repository for development or collaboration.

JavaScript

  • Variables: Used to store and manipulate data in JavaScript.
  • Functions: Blocks of reusable code that perform specific tasks.
  • Conditional Statements: Control the flow of code based on conditions (if, else, switch).
  • Loops: Repeatedly execute code until a certain condition is met (for, while).
  • Objects: Data structures that store related properties and methods.
